Upload ================= --- Options ----------------- * **queue** - the element that will be used as the queue (div) * **showQueue** - if showQueue is set to true, the queue will be shown, if it's false, the queue will be hidden * **queueOptions** - options for the queue * **name** - if set to true, the file name will be shown in the queue * **type** - if set to true, the file type will be shown in the queue * **size** - if set to true, the file size will be shown in the queue * **lastModified** - if set to true, the file last modified date will be shown in the queue * **fileLimit** - the number of files that can be uploaded * **auto** - if auto is true (default), the file will be uploaded as soon as it's added * **debug** - if debug is true (default), then debug info will be logged to the console * **dragDrop** - if true (default), then drag and drop is enabled * **fileType** - the mime-type of the files being uploaded (ex. 'image') * **allowedExts** - defaults to null, meaning there is no extension filter (['jpg', 'jpeg', 'png']) * **dropZone** - this element that will act as the dropzone, must be a div! * **dropZoneText** - 'Drop files here...' * **dropZoneTextSize** - The text size of the dropzone text * **dzDragOverColor** - The background color of the dropzone when the user hovers over it with a file to drop * **inputFile** - The file input element for file upload * **multiple** - whether or not multiple files can be uploaded with the file input * **sizeLimit** - the size limit of the files to upload, 0 is default and means unlimited * **method** - the method to use (i.e. 'post' or 'put') * **action** - the action to take, 'upload.php' is default * **onComplete** - callback for when the request is complete (will always be called, regardless of sucess or error) * **onSuccess** - callback for when the request is completed successfully * **onError** - callback for when the request is completed but there is an error * **beforeSend** - callback for before the request is sent
